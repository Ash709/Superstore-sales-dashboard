# 📊 Superstore Sales Dashboard (MIS Project)

This is an MIS-based Excel dashboard created using PivotTables, charts, and slicers to analyze sales performance and support business decision-making.

## 🔹 MIS Features:

* Data cleaning and structuring
* KPI tracking (Sales, Target, Orders, AOV)
* PivotTable-based analysis
* Trend analysis (Monthly, Weekly, Yearly)
* Interactive dashboard using slicers

## 🔹 Tools Used:

* Microsoft Excel
* PivotTables
* Data Analysis
* Dashboard Design

## 🔹 Outcome:

The dashboard helps management quickly identify trends, top-performing products, and areas needing improvement.
